# onakaitai.org
[README-JP](README-jp.md)

## Overview
"[onakaitai.org](onakaitai.org)" is a website for people with stomach pain. This site displays random images of people experiencing stomach pain. No information is provided about stomach pain.

### Stomach Pain
Stomach pain is an unpleasant sensation experienced in daily life.

### Stomach Pain Is Tough
Stomach pain brings physical discomfort to individuals.

### Advocate for All Those with Stomach Pain
"onakaitai.org" aims to support people with stomach pain. Through randomly displayed images, it empathizes with the feelings of those experiencing stomach pain.
